# Discover Pakistan - Historical & Cultural Cities Website

## Project Overview
A fully responsive educational website showcasing the history, culture, and traditions of 12 major cities of Pakistan.

## Website Structure
- **index.html** - Home page with hero section and city cards
- **cities.html** - All cities overview page
- **styles.css** - Complete CSS styling with Pakistani cultural colors
- **Individual city pages** - 12 detailed pages for each city

## Cities Included
1. Lahore - Heart of Pakistan
2. Karachi - City of Lights
3. Islamabad - Capital City
4. Faisalabad - Manchester of Pakistan
5. Multan - City of Saints
6. Peshawar - Gateway to Khyber Pass
7. Quetta - Fruit Garden of Pakistan
8. Hyderabad - Paris of Sindh
9. Sukkur - City on River Indus
10. Bahawalpur - City of Nawabs
11. Sialkot - Sports Capital
12. Dera Ghazi Khan - Heart of Siraiki Culture

1. Create "images" folder in the project directory
2. Add the required city images
3. Open index.html in any modern web browser
4. Navigate through the website using the menu
